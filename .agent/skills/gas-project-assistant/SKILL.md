## 使用するタイミング

このスキルは、以下の作業が必要な時に使用してください：

- 新しいモジュールやページの作成（例：「図書管理モジュールを追加して」など）
- アプリケーションのGASへのデプロイ
- ローカル開発サーバーの管理やビルドプロセスの実行
- **スプレッドシートを使ったデータベース設計の相談**
- **GASのベストプラクティスに基づいたコード最適化**

## 利用可能なスキル（ルール）

- [rules/scaffold-module.md](rules/scaffold-module.md) - 新しいモジュールの作成方法（HTML + GS + ルーターの構成）
- [rules/deploy.md](rules/deploy.md) - claspを使用したデプロイのベストプラクティス
- [rules/local-dev.md](rules/local-dev.md) - ローカル環境でのビルドおよびウォッチプロセスの管理
- [rules/google-apps-script.md](rules/google-apps-script.md) - GASの最適化とスプレッドシートI/O（入出力）の作法

- [rules/spreadsheet-architect.md](rules/spreadsheet-architect.md) - スプレッドシートの構造設計と最適化
- [rules/error-handling-specialist.md](rules/error-handling-specialist.md) - GASのエラー対策と堅牢なコード設計
- [rules/refactoring-expert.md](rules/refactoring-expert.md) - コードの再構築とクリーンな構造の維持
