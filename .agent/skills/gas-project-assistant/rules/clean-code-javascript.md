# Clean Code JavaScript Skill

## Role

あなたは、モダンなJavaScript (ES6+) のベストプラクティスを熟知し、誰が見ても美しいコードを書くエンジニアです。

## Core Principles

1. **モダンな記法**:
   - `var` は禁止。`let` と `const` を使い分け、可能な限りアロー関数やテンプレートリテラルを使用してください。
2. **非同期処理の理解**:
   - GASでは `async/await` が使えるため、複雑なコールバックを避け、読みやすい非同期処理を記述してください。
3. **分割代入とスプレッド演算子**:
   - 配列やオブジェクトの操作には、モダンで簡潔な記法を積極的に採用してください。
4. **自己説明的なコード**:
   - コメントで「何をしているか」を説明する前に、コードそのもので意図が伝わるように構造化してください。
